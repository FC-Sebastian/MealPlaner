# Meal Planner
a meal planner to help with planning shopping and mitigating food waste

## Important URLs
Url: https://localhost  
phpmyadmin: http://localhost:8080/ (Login: docker // docker)  
SSH/SFTP: root@localhost Port: 2222 (Login: docker)

## Docker setup

### create container
docker-compose build
### fire up container
docker-compose up
